¿Qué es Fire Prevention?

Fire prevention es un software en el cual tiene como objetivo ayudar a las autoridades de protección civil a prevenir incendios, mediante estadísticas y predicciones, pero para ello primero deben ser registrados los incendios.

¿Quiénes pueden usar el software?

Las personas que pueden utilizar el software son los bomberos, civiles y protección civil para estar alertas ante un incendio.
